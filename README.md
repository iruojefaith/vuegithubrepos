# GitHub User Repositories Searcher

This is a Vue.js 3 application for finding public repositories of GitHub users.

## Stack

- Vue.js 3
- Vuex 4
- Vue-Router
- Axios
- Bootstrap 5
- Bootstrap Icons
- SCSS
- time-ago library


## Installation
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
